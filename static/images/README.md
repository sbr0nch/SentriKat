# SentriKat Branding Assets

Place your branding images in this folder:

| File | Size | Usage |
|------|------|-------|
| `favicon.ico` | 32x32, 16x16 | Browser tab icon |
| `favicon-128x128.png` | 128x128 | Default logo in app |
| `logo.png` | 200x200 | Large logo |
| `logo-dark.png` | 200x200 | Logo for dark theme (optional) |

## Favicon Generation

You can generate all favicon sizes from a single image using:
- https://realfavicongenerator.net/
- https://favicon.io/

## Usage in App

The app automatically looks for:
- `/static/images/favicon.ico` - Browser favicon
- `/static/images/favicon-128x128.png` - Default branding logo
